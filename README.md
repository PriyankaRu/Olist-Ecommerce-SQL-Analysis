# Olist Brazilian E-commerce SQL Analysis

SQL analysis of Brazilian e-commerce sales, customer behavior, and product performance using the Olist dataset.

## 📊 Dataset
- **customers**: Customer information and location
- **orders**: Order data with timestamps
- **order_items**: Products in each order
- **products**: Product metadata
- **order_payments**: Payment information

## 🔗 Dataset Link:
https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce

## 🔍 Analysis Includes
✅ Total revenue and monthly trends
✅ Product performance (sales volume & revenue)
✅ Customer behavior and spending patterns
✅ Payment methods and installment usage
✅ Geographic distribution of orders
✅ Repeat vs one-time customer analysis

## 📁 Files
- `SQL Project - Brazilian E-commerce Sales & Customer Behavior Analysis.sql` - All 12 queries

## 🛠️ Tools Used
- PostgreSQL
- SQL
- Olist Brazilian E-Commerce Public Dataset (Kaggle)
